
 # Basic Linux Commands


- **`tree`**: ساختار دایرکتوری‌ها و فایل‌ها رو به‌صورت درختی نمایش می‌ده.
    
- **`find`**: برای جستجوی فایل‌ها و دایرکتوری‌ها بر اساس نام، نوع، زمان، اندازه و… استفاده می‌شه.
    
- **`grep`**: داخل فایل‌ها یا خروجی دستورات، دنبال یک الگو یا متن خاص می‌گرده.
    
- **`chmod`**: سطح دسترسی فایل‌ها و دایرکتوری‌ها (خواندن، نوشتن، اجرا) رو تغییر می‌ده.
    
- **`ps`**: لیست پردازش‌های در حال اجرا رو نمایش می‌ده.
    
- **`top`**: وضعیت زنده‌ی پردازش‌ها و مصرف CPU و RAM رو نشون می‌ده.
    
- **`kill`**: برای ارسال سیگنال (معمولاً خاتمه) به یک پردازش استفاده می‌شه.
    
- **`htop`**: نسخه‌ی پیشرفته‌تر و تعاملی‌تر `top` با رابط کاربری بهتره.
    
- **`$PATH`**: متغیری که مسیرهایی رو مشخص می‌کنه که شل برای پیدا کردن دستورات اجرایی جستجو می‌کنه.
    
- **`alias`**: اسم کوتاه یا میانبر برای یک دستور یا چند دستور تعریف می‌کنه.
    
- **`source`**: دستورات داخل یک فایل رو در همان شل فعلی اجرا می‌کنه (بدون باز شدن شل جدید).
    

# Summary of the above commands with examples

|دستور|توضیح کوتاه|مثال|
|---|---|---|
|`tree`|نمایش ساختار فایل‌ها و پوشه‌ها به‌صورت درختی|`tree /home/user`|
|`find`|جستجوی فایل‌ها و پوشه‌ها بر اساس شرط|`find . -name "*.txt"`|
|`grep`|جستجوی یک متن یا الگو داخل فایل‌ها|`grep "error" log.txt`|
|`chmod`|تغییر سطح دسترسی فایل یا پوشه|`chmod +x script.sh`|
|`ps`|نمایش پردازش‌های در حال اجرا|`ps aux`|
|`top`|نمایش زنده وضعیت پردازش‌ها|`top`|
|`kill`|ارسال سیگنال برای متوقف کردن پردازش|`kill 1234`|
|`htop`|نسخه‌ی پیشرفته و گرافیکی‌تر `top`|`htop`|
|`$PATH`|مسیرهای جستجوی دستورات اجرایی|`echo $PATH`|
|`alias`|ساخت میانبر برای دستورات|`alias ll='ls -l'`|
|`source`|اجرای فایل در شل فعلی|`source ~/.bashrc`|
